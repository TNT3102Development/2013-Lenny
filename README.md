# 2013-Lenny
Code written in NI LabVIEW 2014 for 2013 "Ulitmate Ascent" robot using cRIO. No autonomous code, only teleop.
http://www.tnt3102.org/lenny

NI LabVIEW 2014 with 2014 FRC Update Suite is required to use the code, as well an FRC Driver Station version 2014 or 2015+ set with 2014 protocol for driving the robot.
